A Verification Prompt That Helps Reduce AI Hallucinations

Published on July 2, 2026 • By Tweaknook Editorial

No prompt can guarantee that an AI response is completely accurate. A better goal is to make unsupported claims easier to detect before the output is used in an email, report, spreadsheet, or client document.

Why Simple Prompts Produce Confident Errors

A request such as “summarize this topic and give me the facts” does not define what counts as evidence. The model may combine correct information with assumptions, outdated details, or invented citations. A verification prompt adds clear rules for sources, uncertainty, and review.

A Practical Verification Prompt

Copy the following prompt and replace the bracketed sections:

[TASK]
Answer this question: [INSERT QUESTION]

[SOURCE RULE]
Use only the information contained in the material I provide below.
Do not use outside knowledge unless I explicitly ask for it.

[ACCURACY RULES]
1. Do not guess missing facts.
2. Label uncertain statements as "Unverified."
3. For every important claim, identify the supporting page, section, or quoted phrase.
4. If two sources conflict, show both statements instead of choosing one silently.
5. If the source does not contain the answer, say: "The provided material does not answer this question."

[OUTPUT]
A. Direct answer
B. Evidence used
C. Unverified or missing information
D. Final verification checklist

[SOURCE MATERIAL]
[PASTE TEXT OR ATTACH DOCUMENT]

How to Use the Prompt

  1. Define the source boundary. Tell the model whether it may use only an attached file or whether outside research is allowed.
  2. Ask for evidence locations. Page numbers, headings, or short supporting excerpts make manual checking faster.
  3. Separate facts from suggestions. Ask the model to label recommendations and inferences instead of presenting them as source facts.
  4. Review the highest-risk claims first. Check names, dates, prices, legal rules, medical statements, calculations, and technical commands.

Example

Suppose a policy PDF is attached and you ask when a requirement begins. A weak response may provide a date without showing where it came from. With the verification prompt, the response should include the date, the relevant page or heading, and a warning when the document is ambiguous.

What This Prompt Cannot Do

  • It cannot make an incorrect source accurate.
  • It cannot guarantee that every citation is matched correctly.
  • It cannot replace testing of code, formulas, or configuration changes.
  • It should not be the only review step for legal, financial, medical, or security decisions.

Important: Treat AI output as a draft. Open the original source and confirm important claims before publishing or acting on them.

A Simple Final Check

Before using the answer, ask three questions: Is the claim present in the source? Does the cited location actually support it? Is any conclusion stronger than the evidence? This short review catches more problems than adding dramatic instructions such as “be 100% accurate.”
