I Wasted 4 Hours Setting Up My New Work PC. Here is the 3-Line Script That Installs Every App in 5 Minutes

Published on July 15, 2026 • By Tweaknook Editorial

Last Tuesday, our IT department finally handed me the keys to a brand-new, high-performance workstation laptop. The hardware specifications were absolutely stunning, but my excitement evaporated the exact moment I booted it up and stared at the completely barren Windows desktop screen. A fresh computer means a massive, mind-numbing administrative chore: initial software installation. To get my workstation back to full operational capacity, I needed nearly twenty different applications. I needed Google Chrome, Slack, Zoom, Microsoft PowerToys, Notion, Spotify, Visual Studio Code, and a dozen other specific utility tools.

Historically, my workflow for setting up a new PC looked like this: open the default web browser, search for "Download Chrome," click the link, download the `.exe` file, run the installer, click "Next" five times, explicitly uncheck the box asking to install a useless web toolbar, and wait for the loading bar to finish. Then, I had to repeat this exact same tedious process for nineteen other applications. You have to navigate to twenty different corporate websites, dodge confusing marketing banners, and physically babysit installation wizards. This manual process typically consumes half of a workday. It is the very definition of low-value, zero-leverage administrative friction. That was the last time I ever set up a computer manually. Today, I use a native, command-line package manager to install my entire software stack simultaneously with a single string of text.

The Security Risks of Manual Web Installers

Beyond the massive waste of time, downloading software manually through a web browser introduces severe corporate security vulnerabilities to your machine. When you type "Download Zoom" into a search engine, you are placing immense, blind trust in the search algorithm. Cybercriminals exploit this human behavior through a tactic called "typosquatting" and search engine poisoning. They buy ad placements at the top of the search results that look visually identical to the official website, but when you click and download the `.msi` or `.exe` installer, it contains embedded ransomware or spyware payloads. By blindly clicking through web portals, you are constantly rolling the dice with your system's operational integrity.

Furthermore, manual installations lead to rapid system degradation. Every graphical installer handles background registry keys and installation directories differently. They often leave behind bloated cache folders and install overlapping background update services that constantly drain your CPU resources. If you have ever wondered why a fresh Windows laptop feels incredibly fast on day one, but feels sluggish and unresponsive six months later, the chaotic nature of manual software installation is a primary culprit. We need a centralized, programmatic approach to software management, similar to what Linux engineers have enjoyed for decades with tools like apt-get, or what macOS developers utilize with Homebrew.

The Secret Built-In Tool: Winget

In 2020, Microsoft quietly revolutionized Windows software management by introducing the Windows Package Manager, commonly known within the engineering community as Winget. It is natively built directly into Windows 10 and Windows 11. Most regular office workers have absolutely no idea it exists because it does not have a flashy graphical user interface or a desktop icon. It lives entirely within the Windows PowerShell command terminal.

Winget completely eliminates the web browser from the installation equation. It connects your computer directly to Microsoft’s official, heavily vetted software repository. When you command Winget to install a program, it securely pulls the official, uncompromised installer package, silently verifies its digital cryptographic hash to ensure the file has not been maliciously tampered with, and executes the installation entirely in the background. There are no "Next" buttons to click. There are no bloatware toolbars to manually uncheck. It is a completely silent, frictionless, and highly secure software pipeline straight from the developer to your local hard drive.

Step 1: Building the Master Automation Script

You do not need to be a software engineer to leverage this tool. We are going to construct a simple text command that will install your entire workstation stack in one fell swoop. To begin, click your Windows Start button, type "PowerShell," right-click the application, and select "Run as Administrator."

In the terminal window, you can search for the exact ID of the software you want by typing: winget search "Google Chrome". The terminal will output the exact Application ID (in this specific case, Google.Chrome). Instead of installing applications one by one, we chain the commands together using the && operator. This operator commands the computer to execute the next installation immediately after the previous one finishes. Copy and paste this exact baseline script into your PowerShell terminal:

winget install --id=Google.Chrome -e --accept-source-agreements && winget install --id=SlackTechnologies.Slack -e && winget install --id=Zoom.Zoom -e && winget install --id=Notion.Notion -e && winget install --id=Microsoft.PowerToys -e

The -e flag stands for "exact," ensuring Winget grabs the precise package without pausing to ask you for clarification. When you hit the Enter key, you can literally take your hands off the keyboard. You will watch the terminal automatically download, cryptographically verify, and silently install every single application on that list. What used to take four hours of babysitting is now completed in five minutes while you go grab a cup of coffee.

Step 2: The Ultimate Global Upgrade Command

The operational power of Winget extends far beyond the initial day one setup. Software maintenance is an ongoing corporate nightmare. You likely have a dozen programs running in your system tray constantly begging you to click "Update Now" at the most inconvenient times during a workday.

Winget solves this globally. Once a month, open your PowerShell terminal and type this exact, simple command:

winget upgrade --all

This single string of text forces the package manager to scan every single application installed on your machine, cross-reference their version numbers against the global Microsoft repository, download every available patch, and silently upgrade all of your software simultaneously in the background. It is the ultimate administrative shortcut for keeping your machine secure and up to date.

Step 3: Cloning Your Machine with Export/Import

If you manage multiple computers, or if you simply want an ironclad insurance policy for the future, Winget offers a flawless cloning mechanism. Once your primary workstation is perfectly configured with all your required tools, open PowerShell and type: winget export -o C:\Users\YourName\Desktop\MyApps.json.

This generates a lightweight text file containing the exact manifest of every program on your machine. If your laptop is ever destroyed, or if you transition to a new company, you simply copy that `MyApps.json` file to the new PC. Run the command winget import -i MyApps.json, and the system will read the manifest and automatically rebuild your entire software environment from scratch.

Reclaim Your Time

Your time is your most valuable professional asset. Spending hours navigating ad-filled websites and clicking through graphical installation wizards is an unacceptable inefficiency for any modern knowledge worker. By embracing the command-line power of Winget, you transform a miserable administrative chore into a frictionless, five-minute automated script. Stop acting as a manual computer technician. Deploy the Windows Package Manager, secure your software supply chain, and permanently reclaim your focus.
