How to Clean Messy Spreadsheet Data with Google Sheets Regex

Published on July 16, 2026 • By Tweaknook Editorial

Google Sheets includes regular-expression functions that can extract or replace text based on a pattern. They are useful for repeated cleanup, but every pattern has limits. Test formulas on a sample before applying them to an entire dataset.

Functions Used in This Guide

  • REGEXEXTRACT returns the first part of a text value that matches a pattern.
  • REGEXREPLACE replaces matching text with another value.
  • REGEXMATCH returns TRUE or FALSE when text matches a pattern.

Google Sheets uses the RE2 regular-expression engine, so some patterns from other programming languages may not work in the same way.

Example 1: Extract an Email Address

Assume cell A2 contains:

Please contact John at john.doe@example.com about the order.

Use:

=IFERROR(REGEXEXTRACT(A2, "[A-Za-z0-9._%+-]+@[A-Za-z0-9.-]+\.[A-Za-z]{2,}"), "Review")

The formula returns the first matching address. IFERROR places Review in rows where no match is found.

Limitations

This practical pattern does not validate every address allowed by formal email standards, and it returns only the first match. Use it for cleanup and review, not as proof that an address exists or can receive mail.

Example 2: Keep Only Phone Digits

Assume A2 contains (555) 123-4567. Use:

=REGEXREPLACE(A2, "[^0-9]", "")

The result is 5551234567. Before importing the value into another system, check country codes, extensions, leading zeros, and required number length.

Example 3: Flag Rows for Review

For a dataset where a valid local phone number should contain exactly ten digits after cleanup, place the cleaned number in B2 and use:

=IF(REGEXMATCH(B2, "^[0-9]{10}$"), "OK", "Review")

This checks only the chosen ten-digit rule. It is not suitable for international numbers unless the pattern is adjusted.

Example 4: Trim Repeated Spaces

To replace runs of spaces with a single space:

=TRIM(REGEXREPLACE(A2, "[[:space:]]+", " "))

Test this formula with line breaks and imported web text because whitespace can vary by source.

A Safer Cleanup Workflow

  1. Keep the original column unchanged.
  2. Place the cleanup formula in a new column.
  3. Add a status column that flags unmatched values.
  4. Filter the status column and review exceptions.
  5. Copy and paste values only after confirming the output.

Common Mistakes

  • Using one pattern for every country or data source.
  • Deleting punctuation before checking whether it separates an extension.
  • Overwriting the original data before reviewing results.
  • Assuming a matching email or phone number is valid.
  • Forgetting that REGEXEXTRACT returns the first match unless capture groups are used.

See Google's official references for REGEXEXTRACT and REGEXREPLACE.
