How to Generate and Email PDF Invoices from Google Sheets

Published on July 13, 2026 • Updated on July 23, 2026 • By Tweaknook Editorial

Google Apps Script can read invoice rows from Google Sheets, fill a Google Docs template, convert the result to PDF, and send it as an email attachment. The script below marks a row as sent only after the email command completes.

Test first: Use your own email address and a sample invoice before sending anything to a client. Google account sending limits also apply.

Step 1: Prepare the Sheet

Create a sheet named Invoices with these headers in row 1:

ColumnHeader
AClient
BEmail
CInvoice Number
DDescription
EQuantity
FUnit Price
GStatus

Step 2: Create a Google Docs Template

Create a Google Doc and add placeholders such as:

INVOICE

Invoice Number: {{INVOICE_NUMBER}}
Date: {{DATE}}
Client: {{CLIENT}}
Description: {{DESCRIPTION}}
Quantity: {{QUANTITY}}
Unit Price: {{UNIT_PRICE}}
Total: {{TOTAL}}

Copy the document ID from the address bar. It is the long value between /d/ and /edit.

Step 3: Add the Apps Script

In Google Sheets, open Extensions → Apps Script. Replace the template ID in the first line and paste:

const TEMPLATE_DOC_ID = "PASTE_YOUR_GOOGLE_DOC_ID_HERE";

function generateAndEmailInvoices() {
  const sheet = SpreadsheetApp.getActive().getSheetByName("Invoices");
  if (!sheet) {
    throw new Error('Sheet named "Invoices" was not found.');
  }

  const values = sheet.getDataRange().getValues();

  for (let rowIndex = 1; rowIndex < values.length; rowIndex++) {
    const [client, email, invoiceNumber, description, quantity, unitPrice, status] = values[rowIndex];

    if (!client || !email || !invoiceNumber || status === "Sent") {
      continue;
    }

    const numericQuantity = Number(quantity);
    const numericUnitPrice = Number(unitPrice);
    if (!Number.isFinite(numericQuantity) || !Number.isFinite(numericUnitPrice)) {
      sheet.getRange(rowIndex + 1, 7).setValue("Check quantity or price");
      continue;
    }

    const total = numericQuantity * numericUnitPrice;
    const templateFile = DriveApp.getFileById(TEMPLATE_DOC_ID);
    const temporaryFile = templateFile.makeCopy(`Invoice-${invoiceNumber}`);

    try {
      const document = DocumentApp.openById(temporaryFile.getId());
      const body = document.getBody();

      replacePlaceholder(body, "INVOICE_NUMBER", invoiceNumber);
      replacePlaceholder(body, "DATE", Utilities.formatDate(new Date(), Session.getScriptTimeZone(), "yyyy-MM-dd"));
      replacePlaceholder(body, "CLIENT", client);
      replacePlaceholder(body, "DESCRIPTION", description || "");
      replacePlaceholder(body, "QUANTITY", numericQuantity);
      replacePlaceholder(body, "UNIT_PRICE", numericUnitPrice.toFixed(2));
      replacePlaceholder(body, "TOTAL", total.toFixed(2));

      document.saveAndClose();

      const pdf = temporaryFile
        .getAs(MimeType.PDF)
        .setName(`Invoice-${invoiceNumber}.pdf`);

      GmailApp.sendEmail(
        email,
        `Invoice ${invoiceNumber}`,
        `Hello ${client},\n\nPlease find your invoice attached.\n\nThank you.`,
        { attachments: [pdf] }
      );

      sheet.getRange(rowIndex + 1, 7).setValue("Sent");
    } catch (error) {
      sheet.getRange(rowIndex + 1, 7).setValue(`Error: ${error.message}`);
    } finally {
      temporaryFile.setTrashed(true);
    }
  }
}

function replacePlaceholder(body, name, value) {
  const safeValue = String(value).replace(/\\/g, "\\\\").replace(/\$/g, "\\$");
  body.replaceText(`\\{\\{${name}\\}\\}`, safeValue);
}

Step 4: Run a Test

  1. Add one test row using your own email.
  2. Click Run and choose generateAndEmailInvoices.
  3. Approve the requested Google permissions.
  4. Confirm that the email contains a PDF attachment.
  5. Confirm that column G changes to Sent.

Important Notes

  • The script uses the spreadsheet's time-zone setting for the invoice date.
  • Rows already marked Sent are skipped.
  • Incorrect quantity or price values are not sent.
  • A temporary Google Doc is moved to Trash after the PDF is created.
  • Review taxes, invoice numbering, currency formatting, and legal requirements for your location.

Common Setup Problems

ProblemWhat to check
Template file cannot be openedConfirm that TEMPLATE_DOC_ID contains only the Google Doc ID and that your account can access the file.
Placeholders remain in the PDFMake sure the template uses the exact names shown in the script, including both braces.
Email is not sentReview the Apps Script execution log, recipient address, account permissions, and daily sending quota.
Incorrect totalsCheck that Quantity and Unit Price are stored as numbers rather than formatted text.
