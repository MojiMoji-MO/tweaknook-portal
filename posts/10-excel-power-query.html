How to Automate Recurring CSV Cleanup with Excel Power Query

Published on July 7, 2026 • By Tweaknook Editorial

Power Query is useful when the same type of CSV report arrives repeatedly and requires the same cleanup steps. Instead of editing each file manually, you define the transformations once and refresh the query when new data is available.

Before You Start

This workflow works best when each CSV uses the same column structure. Make a backup of the original file and test the query with a copy before using it for an important report.

Step 1: Import the CSV

  1. Open Excel and select Data.
  2. Select Get Data → From File → From Text/CSV.
  3. Choose the CSV file.
  4. Confirm the delimiter and character encoding in the preview.
  5. Select Transform Data to open Power Query Editor.

Step 2: Record the Cleanup Steps

Each change appears in the Applied Steps panel. Common cleanup steps include:

  • Removing columns that are not needed.
  • Renaming headings to consistent names.
  • Changing text, date, and number data types.
  • Filtering blank or invalid rows.
  • Splitting a combined column by a delimiter.
  • Trimming extra spaces from text.

Check data types carefully. An automatically detected date or decimal type may be wrong when the file uses a different regional format.

Step 3: Load the Result

Select Home → Close & Load. Excel loads the transformed result into a table while keeping the original CSV unchanged.

Step 4: Refresh with New Data

If the next CSV replaces the previous file at the same path and keeps the same name, use Data → Refresh All. Power Query reads the new source and repeats the recorded transformations.

If every report has a different filename, use the From Folder connector instead. That workflow can combine files with the same structure and refresh when new files are added to the folder.

Common Refresh Problems

Source file not found

The file was moved, renamed, or stored on a drive that is not connected. Open Data Source Settings or edit the Source step to select the correct location.

Column not found

The latest CSV changed a heading. Compare the new file with the sample used to create the query and update the affected step.

Numbers load as text

Check the delimiter, decimal separator, and locale used by the Changed Type step. Convert the column only after confirming how the source represents numbers.

Safe Operating Checklist

  • Keep untouched source files in an archive folder.
  • Review row counts before and after filtering.
  • Check a few sample records after each refresh.
  • Document expected column names and data types.
  • Do not assume a refresh succeeded just because no error window appeared.

Microsoft documents Power Query as Excel's data connection and transformation technology. See About Power Query in Excel for current availability and features.
