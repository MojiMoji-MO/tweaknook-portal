How to Free Up Disk Space with Built-In Windows Tools

Published on July 3, 2026 • By Tweaknook Editorial

Two months ago, I was leading a critical quarterly review meeting with the executive team. I had my screen shared on the conference room projector, ready to walk them through a massive data model. I clicked to open the Excel file, and my mouse cursor turned into that dreaded spinning blue circle. Then, a highly embarrassing notification popped up in the bottom right corner of my screen: "Your disk is almost full. Save your work to avoid losing data."

The entire laptop completely locked up. I had to force-reboot the machine while five directors stared at me in awkward silence. When I finally got back to my desk, I frantically started deleting old PDF reports and heavy images, but it barely made a dent. My C: drive was bleeding red. That is when I discovered the harsh truth: the real storage killers are not your work files. They are the invisible system cache files that Windows secretly hoards over time.

The Trap of Third-Party "PC Cleaners"

When most office workers experience a slow computer, they immediately go to Google and download a free "PC Optimizer" or "Junk Cleaner" software. I used to do this too. I highly advise you to stop doing this immediately.

Some third-party cleanup tools add background processes, advertising, or aggressive cleanup options. Windows already includes tools that can remove temporary files and manage storage without installing additional software.

Method 1: Nuking the Hidden Temp Folder

Every time you open an application, browse the web, or extract a ZIP file, Windows creates temporary files. The problem is that Windows is terrible at cleaning up after itself. These files pile up for years, eating away at your precious SSD space and slowing down file indexing.

The steps below remove temporary files. Review the selection before deleting, and skip any file that Windows reports as currently in use.

  1. Hold down the Windows Key and press R to open the Run dialog box.
  2. Type exactly %temp% into the box and hit Enter.
  3. A folder will open containing hundreds or thousands of weirdly named files.
  4. Press Ctrl + A to select every single file in that folder.
  5. Press Shift + Delete to permanently bypass the recycle bin and obliterate them.

If a warning pops up saying "The action can't be completed because the file is open in another program," just check the box that says "Do this for all current items" and click Skip. The amount of space recovered depends on update files, browser caches, and how long the device has been in use.

Method 2: Automating with Storage Sense

Manually deleting temp files is great for emergencies, but I hate doing manual maintenance. Thankfully, Windows 10 and 11 have a brilliant, hidden feature called Storage Sense that does this automatically in the background. Most IT departments forget to turn this on when they issue new laptops.

Maintenance Type Third-Party Cleaners (Avoid) Windows Storage Sense (Use)
Cost Usually requires a paid subscription. Included with Windows.
Resource Usage Heavy. Drains laptop battery life. Zero. Runs silently during idle time.
Safety Risk High. Known to break registry keys. Completely safe and officially supported.

To activate it, hit the Windows key, type "Storage Settings," and hit Enter. Toggle the switch for Storage Sense to the "On" position. Click on "Configure Storage Sense" right below it. I set mine to run Every Week, and I command it to automatically delete files in my Recycle Bin and Downloads folder if they have been sitting there for over 30 days. It can reduce repeated low-disk warnings, but it does not replace regular storage checks or backups.

Method 3: The Advanced Command Prompt Cleanup

If you have run major Windows Updates recently, your system is likely holding onto massive gigabytes of old installation files just in case you want to downgrade. To wipe these out safely, we use a slightly more advanced trick. You will feel like a hacker doing this, but it is incredibly easy.

Click your Start menu, type cmd, right-click on "Command Prompt," and select "Run as administrator." Once the black screen appears, type this exact command and hit Enter:

cleanmgr /sageset:1

This opens a special, hidden version of the Disk Cleanup tool with advanced options. Check every single box in the list (especially "Windows Update Cleanup") and click OK. Then, go back to your black command prompt screen and type:

cleanmgr /sagerun:1

Sit back and let it run. It might take about ten minutes, but this specific command digs deep into the system architecture and safely removes massive update remnants that standard disk cleanup cannot touch.

Final Thoughts

Since implementing this quick three-step routine, my laptop boots up faster, Excel stops freezing during large data merges, and I never have to worry about crashing during a presentation. Take ten minutes this Friday afternoon to run these native tools. Your machine will thank you, and you will never need to install questionable cleaning software ever again.
